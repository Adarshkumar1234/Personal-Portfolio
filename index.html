Project Idea

We are going to build a platform on which every Farmer can buy or order any goods(Urea, pesticides, Insecticide, fertilizers) or item(like water pump) that they need in their day to day life to perform the best agriculture practice at a minimal price or rate.

Domains

This website has 3 use cases or users:- 1. Admin:-It can handle all the website activities like making sub-admin by giving website access to any person to divide the workload. 2. Dealer:-Let say if any dealer or shopkeeper wants to sell their goods like Urea, pesticides, Insecticide, fertilizers) or any electric shopkeeper who sale different types of equipment of farming like water motor etc can register by filling in useful information like Name, Gender, Address, Age, GST number, Display Photo. After that, he can simply start uploading images of their products with minimum Price and start selling. 3. User(Farmer):- Now if any farmer visits this website then he can see all the products listed by different vendors and simply add them to their cart and can order them. The main benefit of this website is they don’t have to pay any amount before home delivery.

Technology Used

We are going to use the latest industrial technologies or industrial practices for a better user experience. So any platform or website has three parts:- • Frontend:- we will use HTML, CSS for Designing and React(a JavaScript Library) for making this website lighting speed so that every page will open before we blink. • *Backend:-We will use Node.js(a Javascript backend framework) and Express(a node.js framework)for all backend stuffs like for making APIs for handling requests. Since we are using Node.js the end-user will get a response very fast • *Database:- We will use MongoDB as a database because it will serve better experience to us and it is easy to use.
